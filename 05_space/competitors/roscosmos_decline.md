# Roscosmos Decline
**정보 신선도:** 🟢  |  **최종 갱신:** 2026-04  |  **다음 갱신:** 2026-07

## 2026년 4월 현재 상태
- 2023-08-20 Luna-25 lander는 maneuvering engine 비정상 작동(84초 계획 → 127초 실제)으로 달 표면에 충돌, Soviet Luna-24(1976) 이후 첫 러시아 달 임무가 실패로 종료됐다.
- Angara A5는 2024-04-11 Vostochny 최초 발사 성공 전까지 두 번의 aborted 시도를 겪었고, 2026-04 기준 누적 A5 임무는 전량 test flight 수준으로 여전히 Proton 대체가 완성되지 않았다.
- 2025 기준 러시아 연간 orbital 발사는 1961 이후 최저 수준으로 떨어졌다. Soyuz-5 첫 발사는 2026-03 말 Baikonur로 공식 예정됐으나 지연 여부는 실행 직전까지 불확실했다.
- 2025-12 Roscosmos의 Oleg Orlov는 ROS(Russian Orbital Service Station)를 ISS의 일부로 재조립 후 2030년 deorbit 전 분리해 독립 운용한다는 수정안을 제시했다. 기존 "2024 이후 탈퇴" 스탠스는 국제 협력 유지 쪽으로 일부 완화됐다.
- 제재·공급망 제약으로 상업 발사 시장 점유는 2014 이후 지속 감소, Soyuz 생산량 감소와 엔진 부품 수급 난이 구조적 문제로 남아 있다.
- 현재 상태 해석:
  - 확인된 사실: Luna-25 실패, Angara A5 지연, Soyuz-5 지속 연기는 러시아 발사체·탐사 capability 쇠퇴의 증거다
  - 확인된 사실: ROS를 통한 독립 궤도 인프라는 선언됐지만 자금·부품 제약이 실행 불확실성을 키운다
  - 이 레포의 추론: Roscosmos는 2035년까지 Soyuz·ISS 의존에서 완전 탈피하지 못하고 CNSA·ILRS 하위 파트너로 자리할 가능성이 높다

## 물리적/구조적 한계
- 극복된 것:
  - Soyuz 발사체 장기 신뢰성 운용
  - ISS 러시아 세그먼트 유지
  - 기존 군·우주 제조 기반 유지
- 미해결:
  - 고성능 반도체·회피 항법 부품 수급
  - Angara A5 cadence 확보
  - Soyuz-5 실증
  - ROS 자금·부품 조달
  - 국제 상업 발사 시장 재진입
- 극복 시나리오:
  - CNSA 파트너십을 통한 기술 공유와 중국산 서브시스템 의존이 현실적 대안

## 기술 현실론 보정
- 낙관론 측 근거:
  - 러시아는 여전히 독자 유인발사 capability(Soyuz)와 ISS 세그먼트 운용 경험 보유
  - CNSA 파트너십은 구조적 지속가능성 확보 경로
- 현실론 측 반론:
  - 제재·예산·인재 유출은 구조적이며 2035까지 회복 어려움
  - Luna-25 실패는 탐사 프로그램 기반 약화 증거
- 균형 판단:
  - Roscosmos는 2035까지 "쇠퇴하지만 완전 소멸하지는 않는" 경로로, CNSA의 하위 파트너 포지션에서 잔존

## 2035 전망 요약
- Base: Soyuz·Angara 기반 축소된 운용 유지, ROS 초기 배치, ILRS 하위 파트너
- Upside: Soyuz-5와 Angara A5 cadence 안정, ROS 유인 운용
- Downside: ROS 축소/취소, CNSA 파트너십 비대칭 심화, 상업 시장 사실상 퇴출
