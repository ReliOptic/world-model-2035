# Project Scoring Framework
**최종 갱신:** 2026-04

## 목적
이 문서는 `15_opportunity_intelligence/` 인텔리전스 시스템에서 평가되는 모든 기회를 동일한 기준으로 채점하기 위한 마스터 루브릭이다. 도메인이 다르더라도 동일 척도로 비교 가능하도록 10개 항목 × 5점 체계를 단일 기준으로 고정한다.

## 10가지 평가 항목

각 항목은 1-5점으로 평가한다. 1점은 명백히 불리한 조건, 3점은 평균적/중립적 조건, 5점은 가장 유리한 조건이다.

| 항목 | 1점 | 3점 | 5점 |
|---|---|---|---|
| 구조적 필연성 | 일시적 트렌드 | 5년 지속 가능성 | 2035까지 구조적으로 커짐 |
| 지불 의사 | 자선/보조금 의존 | 비용으로 처리 | 전략 예산으로 지출 |
| 긴급도 | 3년 후에나 필요 | 18개월 내 수요 | 6개월 내 예산 결정 |
| 데이터 우위 | 공개 데이터로 충분 | 약간의 우위 | 독보적 접근 or 해석 능력 |
| 실행 가능성 | 수년·대규모 팀 필요 | 3-6개월·소규모 팀 | 8주·혼자 MVP 가능 |
| 방어력 | 누구나 복제 가능 | 6개월 선점 우위 | 데이터·관계·브랜드 해자 |
| 시장 크기 | <$100M | $1-10B | >$10B |
| 타이밍 | 너무 이르거나 늦음 | 적절 | 정확한 변곡점 |
| 규제/정책 순풍 | 역풍 가능 | 중립 | 법제·예산이 밀어줌 |
| 개인 적합성 | 잘못된 분야 | 파트너십 필요 | 리서치·보고서·컨설팅으로 직결 |

## 판정 기준

| 총점 | 판정 | 행동 |
|---|---|---|
| 40-50점 | 우선순위 높음 | 즉시 추진 |
| 32-39점 | MVP 실험 | 8주 스프린트 |
| 25-31점 | 콘텐츠 검증 | 리서치·글쓰기로 시장 반응 확인 |
| <24점 | 보류 | 분기별 재평가 |

## 사용 방법
1. 각 항목을 독립적으로 1-5 점수로 부여한다.
2. 10개 항목 점수를 합산해 총점을 산출한다.
3. 위 판정 기준 표에 따라 행동을 결정한다.
4. 개인 적합성 점수가 1-2점이면 총점에 무관하게 "파트너십 없이 추진 불가"로 분류한다.

## 채점 규율
- 항목별 점수는 추측이 아니라 근거 문장 1개 이상으로 뒷받침해야 한다.
- 동일 도메인 안에서도 기회마다 별도로 채점한다 (도메인 전체에 동일 점수를 부여하지 않는다).
- 분기별 재평가 시 점수 변동이 있는 항목은 변경 사유를 함께 기록한다.
- 기회별 채점 결과는 `15_opportunity_intelligence/` 하위 별도 파일에 보관하며, 본 프레임워크 문서는 점수표가 아니라 기준 정의만 유지한다.
