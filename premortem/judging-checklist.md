# 자가채점 체크리스트 — 제출 전 심사기준 점검

> 심사기준(총 100점)에 맞춰 초안을 스스로 채점한다. 각 항목 약점을 찾아 보강하는 도구. 점수가 아니라 **빈칸이 보이게** 하는 게 목적.

## 통찰력 및 참신성 — 35점
- [ ] 기존 AI 위협론(대량실업·킬러로봇·일자리)을 **넘어선** 실패 지점인가?
- [ ] 심사위원이 "이 각도는 생각 못했다" 할 만한 독창성이 있는가?
- [ ] 흔한 디스토피아 클리셰(스카이넷·감시사회 일반론)로 미끄러지지 않았는가?
- [ ] 탐색 층위(개인/관계/사회) 중 하나가 선명하게 잡혀 있는가?

## 논리적 인과성 — 35점
- [ ] '예견된 실패'와 '원인 진단'의 연결 고리가 **빈틈없이** 설득되는가?
- [ ] 원인이 **2026년에 실재하는** 신호·기술·제도에 묶여 있는가? (먼 미래 가정 아님)
- [ ] 출처/근거가 최소 1~2개 명시됐는가? (코퍼스 링크 또는 원자료)
- [ ] "AI가 발전하면 ○○될 것" 식 비약 없이, 단계별 역추적이 보이는가?
- [ ] 한 장면에 집중해 인과 사슬이 흐트러지지 않았는가?

## 현재적 가치 및 실행력 — 20점
- [ ] 대응 방안이 **누가/무엇을/언제까지**로 구체적인가?
- [ ] 오늘~5년 내 실행 가능한 현실성이 있는가?
- [ ] "인식을 바꿔야 한다" 같은 추상 구호가 아닌가?
- [ ] 실제 제도·정책·기술 접점(예: EU AI Act, NGFS, 교육과정)과 연결되는가?

## 전달 및 표현력 — 10점
- [ ] 1페이지를 처음 보는 사람이 30초 안에 핵심 장면을 잡는가?
- [ ] 제목이 실패를 한 줄로 보여주는가?
- [ ] 공감대를 형성하고 캠페인으로 확장될 표현력이 있는가?
- [ ] 도식/시각 요소가 메시지를 돕는가(장식이 아니라)?

## 형식·자격 (감점·실격 방지)
- [ ] A4 1~2페이지, PDF 변환 완료
- [ ] 필수 3요소 모두 포함
- [ ] 순수 창작물 (타인 저작권·초상권 미침해)
- [ ] 온라인 양식 3줄 요약 준비
- [ ] 보완 자료 외부 링크가 'public' 권한
