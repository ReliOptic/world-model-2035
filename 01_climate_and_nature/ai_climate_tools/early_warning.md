# AI Climate Tools: Early Warning Systems
**문서 성격:** concept memo scaffold  
**정보 신선도:** 🔴  |  **최종 갱신:** YYYY-MM  |  **다음 갱신:** YYYY-MM

## 문서 목적
- 기후·재난 조기경보에서 AI가 들어갈 수 있는 기능 영역을 정리
- `현재 운영 중인 체계`, `실증 단계`, `개념 수준 주장`을 구분하는 기준을 만든다
- 성능 주장보다 운영 통합, 경보 전달, 현장 실행 가능성을 우선 평가한다

## 범위 정의
- 포함: 예측 보조, 이상 탐지, 경보 우선순위화, 취약지역 식별, 다중 데이터 통합
- 제외: 일반 기상 예보 전체, 군사 조기경보, 순수 연구용 모델 성능 비교만 있는 사례
- 단위 구분: 국가 체계, 도시 체계, 보험·민간 인프라 운영 체계

## 2026년 4월 현재 체크 항목
- 어떤 재난 유형에 실제 운영 배치가 있는지 확인
- 입력 데이터가 위성·센서·기상·인프라 운영 데이터 중 어디까지 연결되는지 확인
- 경보 리드타임, 오경보/미탐지, 취약계층 도달률 같은 운영 지표 존재 여부 확인
- 모델 정확도와 실제 의사결정 개선이 분리 보고되는지 확인

## 평가 원칙
- 성능 수치는 동일 조건 비교 여부를 먼저 확인
- `정확도 개선`과 `피해 감소`를 같은 것으로 취급하지 않음
- 저소득 지역과 고위험 지역에서의 운영 가능성을 별도 평가
