# Axioms
**정보 신선도:** 🟢  |  **최종 갱신:** 2026-04  |  **다음 갱신:** 2026-07

## Purpose
This file records the base assumptions that anchor the rest of the repository.

## Core axioms
- Observation capacity is expanding faster than human interpretive capacity.
- AI is becoming part of the sensing stack, not only the analysis layer.
- The limiting factors for technology deployment are often energy, manufacturing, regulation, and institutions rather than model capability alone.
- General-purpose technologies diffuse through existing power structures instead of replacing them overnight.
- Geopolitical fragmentation is not a side effect. It is a central variable shaping technical trajectories.
- Climate and biosphere instability increasingly act as economic and geopolitical variables, not externalities.
- Forecast quality improves when current-state facts, institutional friction, and physical constraints are kept separate from aspiration.

## Use
- Treat these as stable defaults until direct evidence justifies revision.
- Reference this file when a topic forecast drifts into narrative without structural anchors.

## Open questions
- Which axioms remain stable through 2030, and which should become measurable variables instead?
- At what point does machine-derived explanation become operationally useful but cognitively unreadable?
