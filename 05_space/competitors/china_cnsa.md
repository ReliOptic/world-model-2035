# China CNSA
**정보 신선도:** 🟢  |  **최종 갱신:** 2026-04  |  **다음 갱신:** 2026-07

## 2026년 4월 현재 상태
- CNSA는 Tianwen-3 Mars sample return을 `2028년 발사`, `2031년 샘플 귀환`으로 공식화했고, 2025-03-11 국제 payload 참여 공모를 개시했다. 샘플 회수 목표는 `최소 500g`이며 2m drill 기반 sample 채취는 세계 최초 시도다.
- Chang'e 7은 `2026년` 달 남극 water-ice 탐사와 ISRU 기초 검증을 목표로 한다. Chang'e 8은 `2029년` 예정이며 lander·rover·robot을 포함해 ILRS 기초 모델 검증 임무를 수행한다.
- Mengzhou 유인 우주선은 `최대 7명` 탑승 구조로, LEO용 변형은 CZ-10A, 달 임무용 변형은 CZ-10 + Lanyue lander 조합이다. CZ-10A 최초 비행은 2027 전후, crew-rated Mengzhou는 2027 달 임무 준비 단계에 진입한다.
- ILRS는 CNSA·Roscosmos 주도이며 CNSA에 따르면 2035까지 남극 기초 시설(1단계)을 구축하고 2050까지 남극·적도·원거리 네트워크(2단계)를 구성한다.
- Long March 계열: Long March 9(CZ-9)는 reusable super-heavy로 개발 중이며, CZ-5B/6A는 주력 중대형 발사체다. 2025년 중국 전체 orbital 발사는 60+ 회 규모로 미국 다음이다.
- 현재 상태 해석:
  - 확인된 사실: 중국은 달 남극 프로그램과 Mars sample return 양축을 동시 운용하는 유일한 비-미국 체계다
  - 확인된 사실: ILRS는 2035 기초 단계까지 실행 가능한 구조로 진행 중이다
  - 이 레포의 추론: 2030년대 초 중국은 crewed 달 착륙 시도 단계에 진입하며, Mars sample return에서 미국보다 먼저 실제 샘플을 귀환시킬 가능성이 구조적 추세다

## 물리적/구조적 한계
- 극복된 것:
  - 중형·대형 발사체 안정 운용(CZ-5/6/7/8)
  - Tiangong 독자 유인 우주정거장 연속 운용
  - 달 남극 착륙 경험(Chang'e 6 샘플 귀환 2024)
- 미해결:
  - 초대형 재사용 발사체(CZ-9) 실증
  - 유인 달 착륙 시스템 통합
  - Mars sample return EDL·귀환 모듈 실증
  - 장기 ILRS surface power
- 극복 시나리오:
  - 국가 주도 자원 배분과 민영 상업 발사체 공존이 cadence 증가를 이끌고, ILRS 국제 파트너 확대가 정치적 정당성 확보

## 기술 현실론 보정
- 낙관론 측 근거:
  - 중국은 공식 로드맵의 실제 집행률이 높고, 각 미션이 순차적으로 실증됐다
  - ILRS·Mars sample return 모두 독립 실행 가능한 자본·산업 기반 확보
- 현실론 측 반론:
  - CZ-9 재사용 로켓과 유인 달 HLS 통합은 검증 전
  - ILRS 2단계(2050)는 2035 관점에서 먼 약속
- 균형 판단:
  - 2035까지 중국은 우주 분야에서 미국 다음 또는 일부 영역(Mars sample return)에서 앞서는 체계로 정착할 가능성이 높다

## 2035 전망 요약
- Base: Chang'e 7·8 완료, ILRS 1단계 기초 완공, Tianwen-3 샘플 귀환, 최초 유인 달 착륙
- Upside: ILRS 국제 파트너십 확대, CZ-9 초기 운용, Mars sample return 세계 최초
- Downside: CZ-9 지연과 유인 달 cadence 제한으로 ILRS가 느린 진척
