How to read this

Base / Upside / Downside / Current implied를 먼저 보고, divergence와 signal을 마지막에 읽는다.

Base

Base는 현재 관측 가능한 제약을 반영한 정상 경로다.

Upside

Upside는 병목 완화가 빨라지는 경로다.

Downside

Downside는 병목이 오래 지속되는 경로다.

Current implied

Current implied는 현재 시장·정책·투자 기대가 암시하는 경로다.

Deterministic output

FLP-0.1

fundamental_score = min(
  demand_score,
  capacity_score,
  finance_score
)

fundamental_line =
  unconstrained_demand_path * fundamental_score

divergence =
  (current_implied - fundamental_line) / fundamental_line
